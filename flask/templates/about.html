Introduction

This project aims to identify and analyze the key factors contributing to student success in academic settings. By leveraging a dataset of over 6,000 records and 20 features, including hours studied, attendance, and parental involvement, we use predictive modeling and clustering techniques to uncover actionable insights.

Data

Dataset: Student Performance Factors
Size: Over 6,000 records
Source: Student Performance Dataset
Attributes: Hours studied, attendance, motivation levels, parental involvement, and more.

Objectives

  • Perform exploratory data analysis to uncover trends in student performance.
  • Build predictive models using regression and classification techniques to forecast exam outcomes.
  • Visualize key findings through interactive dashboards and static graphics.
  • Apply clustering methods to group students based on performance and study habits.
