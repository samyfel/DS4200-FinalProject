Continuous Features

The correlation of the continuous features to performance (exam scores) demonstrates the strength of a relationship. While only hours studied and attendance seem to correlate significantly (see Figure 2 and Table 2), instead of discarding features based just on correlation values, a more statistically powerful way to determine relevant features is through significance testing. To assess which continuous features create significant differences in student performance, the Pearson-Correlation Coefficient Test can be used. IV. Multiple Linear Regression

A multiple linear regression model was fitted to the data. All points, where the score is within 4 standard deviations from the mean were kept. This was done for the following reasons/assumptions:

  1. Large outliers cause poor performance in data fitting
  2. By removing large outliers, we can create a model that is more robust to typical scores
  3. Outliers, which typically happen to the right, indicate that some students perform better than the expected population. Other factors that are not considered (such as outside education and IQ), could cause these differences, and are not reflective of the true population given the features involved.

The features input (after removing irrelevant features) are: Learning Disabilities, Parental Involvement, Access to Resources, Teacher Quality, Internet Access, Peer Influence, Distance from Home, Parental Education Level, Extracurricular Activities, Motivation Level, Family Income, Tutoring Sessions, Hours Studied, Physical Activity, Previous Scores, and Attendance.

All categorical features were transformed using one-hot encoding to convert to a numerical feature. This gave a model with a MAE of 0.31 (which indicates the expected absolute error between the predicted value and expected value) and a R-squared value of 0.9335 on the test set, which indicates a great fit of the model to new data.
